# NewIndia Backend

Node.js backend server for the NewIndia platform.

## Environment Setup

1. Create a `.env` file in the server directory:

```env
PORT=5002
```

2. Install dependencies:

```bash
npm install
# or
yarn install
```

3. Start the development server:

```bash
npm run dev
# or
yarn dev
```

## Project Structure

```
server/
├── src/
│   ├── controllers/   # Request handlers
│   ├── routes/       # API routes
│   └── index.js      # Entry point
└── README.md
```
